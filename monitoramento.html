Como recebo os alertas?

Os alertas chegam direto no seu WhatsApp, com o nome do sensor, a temperatura registrada e um emoji de alerta para fácil identificação.

Preciso instalar algum aplicativo?

Não. O painel funciona direto no navegador do celular ou computador, sem necessidade de instalar nada.

Cada cliente tem acesso separado?

Sim. Cada cliente possui login e senha próprios, acessando apenas os seus sensores. A RefrifastGo tem acesso geral para supervisão.
